# xcrdtbl

This is an experimental library for testing out CRDTs as I learn about them. It
provides implementations for some of the basic, state-based CRDTs. In the
future, I'll likely add a text CRDT as well.

You can see it in use on GitHub where I'm dogfooding it on a task management
CLI/TUI tool called [eph](https://github.com/lucasmcclean/eph).

If you are interested in Rust or CRDTs and want to talk or have questions, feel
free to reach out! I'm also happy to take any feedback or advice.

The non-experimental version is called `crdtbl` (this version is currently
empty).
